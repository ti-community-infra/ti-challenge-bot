PR 模板

关联语义

使用 GitHub 关联语义 #xxxx，我们则会检查当前关联 issue 是否为挑战赛题目，然后进行判断评分。

格式要求

- Issue Number: #xxx

- Issue Number: close #xxx

注意事项：

  • 1.目前的关联格式只支持 #xxx，虽然粘贴 issue 链接也会被 GitHub 识别，但目前还未做到这个关联语义支持。
